Stream Deck Plugin

AC EVO Telemetry Support

Live Assetto Corsa EVO telemetry and controls for Stream Deck on Windows.

Requirements

Windows 10 or newer, Stream Deck 6.5 or newer, and Assetto Corsa EVO running with native shared memory available.

Controls

Control buttons send keyboard input to EVO. Set the same keys in EVO and in each Stream Deck action's property inspector.

Telemetry

The plugin reads EVO shared memory directly. No Fanatec software, Mosquitto, MQTT, or Python bridge is required.

Troubleshooting

  • If a key shows NO DATA, start EVO and load into a session.
  • If a key shows STALE, restart Stream Deck after EVO is running.
  • If a control button does not work, verify the key binding in EVO and in the Stream Deck property inspector.
  • Some EVO fields are not exposed or are not populated by the game yet. Unsupported flags and dead fields are intentionally omitted from the main action palette.

Reporting Issues

Open an issue on the project repository with your Stream Deck version, AC EVO version, the car/session you tested, and what the local diagnostics endpoint reports at http://127.0.0.1:8765/health.
